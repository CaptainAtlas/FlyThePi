Hardware: Raspberry Pi, GPS Module

Software: Raspbian, GPSD, BaseMap

We used a Raspberry Pi in comination with a USB GPS module in order to gather raw GPS data. Using GPSD (a GPS daemon), we were able to see the raw GPS data in set refresh intervals. Using python's csv module, matplotlib, and basemap, we were able to store this data and visualize it in a useful manner.
